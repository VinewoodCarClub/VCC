# Vinewood Car Club - Web

## Soubory
- `index.html` - Hlavní stránka
- `trate.html` - Stránka s tratěmi (placeholder)
- `vozidla.html` - Stránka s vozidly (placeholder)
- `style.css` - Všechny styly
- `script.js` - Interaktivita (menu)

## Jak nahrát na GitHub Pages

1. **Nahraj soubory do svého repozitáře:**
   - Všechny soubory (HTML, CSS, JS) nahraj do hlavního adresáře repozitáře
   - Banner obrázek nahraj jako `banner.jpg` (nebo uprav cestu v CSS)

2. **Aktivuj GitHub Pages:**
   - Jdi do Settings tvého repozitáře
   - V levém menu klikni na "Pages"
   - V sekci "Source" vyber "Deploy from a branch"
   - Vyber branch "main" a složku "/ (root)"
   - Klikni "Save"

3. **Web bude dostupný na:**
   - `https://tvoje-uzivatelske-jmeno.github.io/nazev-repozitare/`

## Banner
- Banner obrázek nahraj jako `banner.jpg` do stejné složky
- Nebo uprav cestu v `style.css` na řádku 83:
  ```css
  background: url('banner.jpg') center/cover no-repeat;
  ```
